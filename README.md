Bugatti简介
---
Bugatti是专为解决开发、运维人员在项目打包、发布、部署、项目依赖、跨环境配置，健康监控等问题统一处理平台。
使用Bugatti可以快速的完成各种项目的安装，部署等各种简化发布工作，同时Bugatti还提供了大量的辅助功能

Bugatti解决下列问题
---
- 部署（监控）任意一个大型应用
	- 高效率
	- 最少步骤
	- 安全
	- 可重复
- 确保一致性
- 出现问题时很快发现并排除故障
