项目版本Api V1

注意：目前接口不对外系统开放，仅支持Jenkins。

添加项目版本

POST  http://bugatti2.dev.ofpay.com/v1/project/version
参数 意义 备注
projectName 项目名称 必传，和系统保持一致
groupId 组件名 必传，对应pom.xml中groupId
artifactId 工件名 必传，对应pom.xml中artifactId
version 版本号 必传，对应当前版本
authToken 系统分配的token值 必传，系统双方约定的值
返回： 
    status=200，添加成功，{"r": "ok"}
    status=400，参数格式错误
    status=401，token错误
    status=404，项目不存在
    status=409，版本冲突，{"r": "exist"}
